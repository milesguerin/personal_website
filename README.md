# GitHub Pages site (Minimal theme)

This site uses the GitHub Pages-supported `jekyll-theme-minimal` theme.

## Publish
1. Create a new GitHub repository.
2. Upload all files in this folder.
3. In GitHub, open **Settings â†’ Pages**.
4. Under **Build and deployment**, select **Deploy from a branch**.
5. Choose `main` and `/ (root)`.
6. Save and wait for deployment.

Your site will be available at:
- `https://<username>.github.io/<repo>/` for a project site, or
- `https://<username>.github.io/` if the repo is named `<username>.github.io`.
